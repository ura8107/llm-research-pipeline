# llm-research-pipeline

My use case and sandboxing lab to apply LLMs for empirical social science research.

Use LLMs to make all of your research-related tasks lightning fast!

1. Summarize and reading academic papers
2. Wrangling messy raw data
3. Implementing minimum quantitative analysis
4. Review and format your drafts
5. Slide and poster generation

## Setup
### Prerequisites

- VSCode or Cursor
- Claude Code CLI (I highly recommend to use CLI, instead of IDE extention for usability)
- Codex CLI
